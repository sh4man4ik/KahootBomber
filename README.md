# What is it?

This is a website for spam bots in Kahoot.

# Quick Start

The website can be accessed at https://kahoot-bomber.vercel.app

# Source Code

Don't forget to change **prod** to **dev** in the _getApiLink.ts_ (frontend/src/modules/DataFieldset/helpers) file.

1. Clone the repository:

```
git clone https://github.com/sh4man4ik/KahootBomber.git
```

2. Go to the directory:

```
cd KahootBomber
```

3. Download frontend dependencies (from _KahootBomber_ directory do):

```
cd frontend
npm install
```

4. Download backend dependencies (from _KahootBomber_ directory do):

```
cd backend
npm install
```

5. Launch the website (from _frontend_ directory do):

```
npm run client
```

6. Launch the server (from _backend_ directory do):

```
npm run server
```

# Tech stack

The following technologies are used to develop the website: **TypeScript**, **React**, **Tailwind CSS**, **daisyUI**, **Node.js**, **Express**, **kahoot.js-latest**, **Vite** and **Google Fonts**.
